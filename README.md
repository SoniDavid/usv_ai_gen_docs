AI Generated Documentation about Vanttec's USV. Intended to be a first approach to understanding how the project is divided and each ros2 package's content.

## Note: 
Do not take this approach as real, complete documentation. Only as an initial approach, specially to new software members and other area members who would like to understand a basic approach to software. For a real understament of the project, please read the code in the vanttec_usv repository.
